# Landing Page - جاهزة

مشروع صفحة هبوط جاهز للنشر. الملفات الأساسية داخل `index.html`, `styles/`, `scripts/`, و `assets/`.

تشغيل محلي:
1. افتح `index.html` مباشرة في المتصفح.
2. أو استعمل خادم بسيط: `npx serve .` أو `python -m http.server 8080`.

خطوات بناء (اختياري):
- استخدم أدوات مثل terser وclean-css لعمل minify لملفات js وcss إلى `build/`.
